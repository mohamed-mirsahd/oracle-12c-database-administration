# Oracle 12c Database Administration
Deployed Oracle Database 12c on a Microsoft Azure Virtual Machine, configured pluggable databases (PDBs), implemented user administration, roles, profiles, tablespaces, security controls, schema management, data import, and index creation using SQL Developer and SQL*Plus
## Key Tasks
- Oracle 12c Installation
- Pluggable Database (PDB) Creation
- User Administration
- Role & Profile Management
- Tablespace Management
- Database Security
- Schema Object Management
- Data Import using SQL Developer
- Index Creation

## Technologies
- Oracle Database 12c
- SQL Developer
- SQL*Plus
- Microsoft Azure VM
